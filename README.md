# 左側低估值選股器 Pro Max

這是一個可在手機瀏覽器使用的 Streamlit 選股 App，適合偏左側交易、低估值、分批布局的台股篩選邏輯。

## Pro Max 新增功能

- 財報品質濾網：EPS、獲利成長率、毛利率、營業利益率、ROE、營業現金流、負債比。
- 財報品質資料可用兩種方式：
  - 上傳 `financial_quality_template.csv` 格式的 CSV。
  - 勾選 Yahoo Finance 補財報品質資料；資料可能缺漏，且全市場掃描會較慢。
- 大盤風險燈號：用加權指數的均線、RSI、20/60 日報酬判斷綠燈、黃燈、紅燈。
- 大盤紅燈處理：只提示、只保留 A/C 類、或停止篩選。
- 產業強弱濾網：產業 20 日平均報酬、產業相對大盤 20 日、產業內上漲比例。
- 部位風險計算器：根據帳戶資金、單筆風險、停損距離，自動估算建議股數、張數與三段式買進張數。
- 觀察名單狀態追蹤：等待試單、已到試單區、二次回測中、站上確認加碼價、跌破停損。

## 原本 Pro 功能

- 自訂本益比上限，例如 `< 20`、`< 15`、`< 30`
- P/B 股價淨值比上限篩選
- 殖利率下限篩選
- 月營收防呆：最近月營收 YoY、累計營收 YoY
- 產業相對本益比：個股本益比可限制低於產業平均一定比例
- 左側型態分類：
  - A 類｜低檔整理型
  - B 類｜急跌反彈型
  - C 類｜基本面錯殺型
  - D 類｜疑似價值陷阱
  - E 類｜結構偏弱
- 價值陷阱警示
- 自動產生交易計畫：試單區、二次回測、確認加碼、停損、壓力、風報比
- 觀察名單 CSV 下載與重新上傳
- 單檔圖表與簡易技術回測

## 資料來源

- 臺灣證券交易所 OpenAPI：上市股票估值、行情、月營收
- 證券櫃檯買賣中心 OpenAPI：上櫃股票估值、行情、月營收
- yfinance：歷史 K 線、加權指數、可選 Yahoo 財報品質欄位
- 使用者上傳 CSV：可補 EPS、ROE、毛利率等財報品質欄位

## 財報品質 CSV 欄位

可直接使用附上的 `financial_quality_template.csv`。建議欄位：

```text
代號,名稱,近四季EPS,EPS年增%,毛利率%,營業利益率%,ROE%,營業現金流,負債比%
```

欄位不一定要全部都有；App 會自動辨識可用欄位。CSV 資料會優先於 Yahoo 補充資料。

## 本機執行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 部署到手機可用的網址

1. 建立一個 GitHub repo。
2. 上傳 `app.py`、`requirements.txt`、`.streamlit/config.toml`、`financial_quality_template.csv`。
3. 到 Streamlit Community Cloud 建立 App。
4. Main file path 填 `app.py`。
5. Deploy 完成後，手機打開網址即可使用。
6. iPhone 可在 Safari 分享選單選「加入主畫面」，就會像 App 一樣開啟。

## 使用提醒

第一次使用建議先把「最多下載 K 線檔數」設在 100～300。確認可以正常跑出資料後，再逐步提高掃描檔數。

若勾選 Yahoo 財報品質資料，速度會比純技術與官方資料篩選慢，建議先搭配較小的 `最多下載 K 線檔數` 測試。

簡易回測只測技術條件，不含歷史本益比、歷史 P/B、歷史月營收，因此不是完整策略回測，只能當作粗略參考。
